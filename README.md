### 技术选型

vue+element-ui+nodeJs+mysql

基于 vue.js，使用 vue-cli4 脚手架，引用 Element UI 组件库搭建的个人博客。

## 实现的功能



> 前台展示页面

![](./assets/home.jpg)

- 用户：登录 注册 退出
- 首页：文章列表(分页)、文章分类、置顶文章
- 留言
- 评论
- 题库

> 后台管理系统
> 系统首页(未完成)
> 统计分析(未完成)
> 文章管理
> 用户管理
> 留言管理
> 题库管理

![](.\assets\admin.jpg)
